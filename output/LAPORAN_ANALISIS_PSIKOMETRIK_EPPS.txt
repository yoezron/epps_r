BAB I: RINGKASAN EKSEKUTIF 
================================================================================ 

1.1 TUJUAN ANALISIS
-------------------------------------------------------------------------------- 

Analisis ini bertujuan untuk:

1. Mengevaluasi properti psikometrik instrumen Edwards Personal Preference
   Schedule (EPPS) menggunakan metode Classical Test Theory (CTT) dan Item
   Response Theory (IRT) modern.

2. Mengidentifikasi kualitas item, reliabilitas, dan validitas konstruk dari
   15 aspek kepribadian yang diukur.

3. Menganalisis struktur hubungan antar aspek kepribadian menggunakan network
   analysis dan Gaussian Graphical Models.

4. Mengatasi masalah ipsativity yang inherent dalam format forced-choice
   melalui Thurstonian IRT.

5. Mengembangkan norma penyekoran yang representatif dan terstandarisasi
   untuk populasi target.

6. Memberikan insight berbasis data untuk pengambilan keputusan HR dan
   assessment talent di PT. Nirmala Satya Development.


1.2 METODOLOGI
-------------------------------------------------------------------------------- 

Analisis dilakukan menggunakan pendekatan multi-method:

A. CLASSICAL TEST THEORY (CTT)
   - Cronbach's Alpha untuk konsistensi internal
   - McDonald's Omega sebagai alternatif yang lebih robust
   - Analisis korelasi inter-item

B. ITEM RESPONSE THEORY (IRT)
   - 2-Parameter Logistic Model (2PL) untuk difficulty dan discrimination
   - 3-Parameter Logistic Model (3PL) dengan parameter guessing
   - Multidimensional IRT (MIRT) untuk 15 dimensi simultan
   - Graded Response Model (GRM) untuk respon polytomous
   - Thurstonian IRT (TIRT) untuk koreksi ipsativity

C. NETWORK ANALYSIS
   - Gaussian Graphical Models untuk struktur partial correlation
   - Centrality measures (Strength, Betweenness, Closeness)
   - Community detection dengan algoritma Walktrap

D. ANALISIS KOMPARATIF
   - Uji t independen untuk perbandingan gender
   - ANOVA untuk perbandingan tingkat pendidikan
   - Effect size calculation (Cohen's d)

E. VISUALISASI
   - 20+ visualisasi berkualitas tinggi (300 DPI)
   - Interactive 3D scatter plots
   - Hierarchical clustering heatmaps
   - Network diagrams

SAMPLE:
  Total Responden  : 6464 orang
  Total Item       : 420 item
  Total Aspek      : 15 aspek kepribadian
  Periode Data     : Januari 2026

--------------------------------------------------------------------------------

   • Aspek dengan skor rata-rata terendah : Heterosexuality (M=5.20, SD=4.53)
   • Variabilitas tertinggi               : Order (Var=20.85)

1.4 REKOMENDASI UTAMA
-------------------------------------------------------------------------------- 

Berdasarkan temuan analisis, berikut rekomendasi utama:

PRIORITAS TINGGI:

1. IMPLEMENTASI SKOR BERBASIS IRT
   → Gunakan skor TIRT atau MIRT untuk mengatasi ipsativity
   → Lebih akurat dibandingkan raw score untuk forced-choice format
   → Estimasi trait level yang tidak bias

2. REVISI ITEM UNTUK ASPEK DENGAN RELIABILITAS RENDAH
   → Deference (Omega = 0.490) perlu item tambahan atau revisi
   → Target minimum: Omega ≥ 0.70 untuk semua aspek
   → Konsultasi dengan expert untuk item development

3. PENGGUNAAN NORMA YANG TEPAT
   → Gunakan norma demografis sesuai populasi target
   → Update norma secara periodik (setiap 3-5 tahun)
   → Pertimbangkan norma spesifik untuk industri/jabatan

PRIORITAS MENENGAH:

4. VALIDASI KONSTRUK LANJUTAN
   → Uji validitas konvergen dengan instrumen lain
   → Uji validitas diskriminan antar aspek
   → Confirmatory Factor Analysis (CFA)

5. ANALISIS DIFFERENTIAL ITEM FUNCTIONING (DIF)
   → Cek bias item antar grup demografis
   → Pastikan fairness dalam assessment

REKOMENDASI JANGKA PANJANG:

6. PENGEMBANGAN SISTEM INTERPRETASI OTOMATIS
   → Dashboard interaktif untuk interpretasi hasil
   → Narrative report generation
   → Integration dengan HRIS

7. PENELITIAN VALIDITAS PREDIKTIF
   → Korelasi dengan job performance
   → Longitudinal study untuk track outcomes
   → ROI calculation untuk talent selection

3.1 15 ASPEK KEPRIBADIAN YANG DIUKUR
-------------------------------------------------------------------------------- 

Edwards Personal Preference Schedule (EPPS) mengukur 15 aspek kepribadian
berdasarkan teori kebutuhan Murray. Berikut adalah deskripsi lengkap:

 1. Achievement (ACH)
    Dorongan untuk mencapai prestasi, menyelesaikan tugas sulit, dan mengungguli orang lain

 2. Deference (DEF)
    Kecenderungan untuk mengikuti saran orang lain, menerima kepemimpinan, dan menghormati otoritas

 3. Order (ORD)
    Kebutuhan akan keteraturan, organisasi, dan struktur dalam kehidupan sehari-hari

 4. Exhibition (EXH)
    Keinginan untuk menjadi pusat perhatian, menceritakan hal lucu, dan menghibur orang lain

 5. Autonomy (AUT)
    Kebutuhan akan kebebasan, independensi, dan melakukan sesuatu dengan cara sendiri

 6. Affiliation (AFF)
    Keinginan untuk dekat dengan orang lain, membentuk persahabatan, dan berpartisipasi dalam kelompok

 7. Intraception (INT)
    Kemampuan memahami perasaan dan motivasi orang lain, menganalisis perilaku

 8. Succorance (SUC)
    Kebutuhan untuk menerima dukungan, bantuan, dan perhatian dari orang lain

 9. Dominance (DOM)
    Dorongan untuk mengontrol, memimpin, dan mempengaruhi orang lain

10. Abasement (ABA)
    Kecenderungan untuk merasa bersalah, menerima kritik, dan menempatkan diri di bawah orang lain

11. Nurturance (NUR)
    Dorongan untuk membantu, merawat, dan menunjukkan simpati kepada orang lain

12. Change (CHG)
    Kebutuhan akan variasi, pengalaman baru, dan perubahan dalam rutinitas

13. Endurance (END)
    Kegigihan dalam menyelesaikan tugas, ketekunan menghadapi kesulitan

14. Heterosexuality (HET)
    Ketertarikan sosial terhadap lawan jenis, membentuk hubungan romantis

15. Aggression (AGG)
    Dorongan untuk mengkritik, menyerang secara verbal, dan mengekspresikan kemarahan


3.2 STRUKTUR ITEM DAN FORMAT FORCED-CHOICE
-------------------------------------------------------------------------------- 

KARAKTERISTIK INSTRUMEN:

• Total Item        : 420 item
• Format            : Forced-choice pairs
• Item per Aspek    : 28-28 item (bervariasi)
• Skala Response    : Binary/multiple choice

IMPLIKASI FORMAT FORCED-CHOICE:

1. IPSATIVITY PROBLEM
   • Skor antar aspek saling bergantung (sum to constant)
   • Korelasi negatif artifisial antar aspek
   • Tidak bisa membandingkan absolute level antar individu
   • SOLUSI: Thurstonian IRT (lihat BAB VII)

2. KEUNTUNGAN FORMAT INI:
   • Mengurangi social desirability bias
   • Forced ranking reveals true preferences
   • Lebih sulit untuk 'fake good'

4.3 PERBANDINGAN CRONBACH'S ALPHA VS McDONALD'S OMEGA
-------------------------------------------------------------------------------- 

McDonald's Omega umumnya lebih robust dibanding Cronbach's Alpha karena:

• Tidak mengasumsikan tau-equivalence (equal factor loadings)
• Lebih akurat untuk data yang tidak perfectly parallel
• Memberikan estimasi reliabilitas yang lebih realistic
