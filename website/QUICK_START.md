## Cara Tercepat Membuka Website

### Windows

1. **Double-click** file `start_server.bat`
2. Browser akan otomatis membuka website di `http://localhost:8000`
3. **Atau** double-click langsung pada `index.html`

### Linux / Mac

1. Buka Terminal di folder `website/`
2. Jalankan: `./start_server.sh`
3. Buka browser dan akses: `http://localhost:8000`
4. **Atau** double-click langsung pada `index.html`
